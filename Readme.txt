Readme File:-
The github link for entire uploaded project - https://github.com/jui-git/jui_meto
Dependencies-

1. jvm 8.0 and jre 8.0 properly installed and configured
2. eclipse ide
3. ifogsim simulator




1. Download the simulator from https://github.com/Cloudslab/iFogSim.

2. It contains the following repositories-
    .idea
    .dataset
    .jars
    .output
    .results
    .src
    .topologies
    .gitignore
 

3. Open eclipse. Go to the new tab and create a java project tab. Name your project and import the downloaded ifogsim files. Click Finish.

4. In the left bar you will see the ifogsim simulator files.

5. Import java code under src folder of ifogsim. In the src folder go to org.fog.test.perfeval. Import import jui_meto.java code file .Right click on the code file and run the code.

6. Output will be displayed in the bottom panel of eclipse.

7. Note - You can also run the code on any java compiler or online gdb compiler
